# RigidOJ
RigidOJ is an online judge system for ACM & its training process.

## Features

- Import problems from popular ACM sites.
- Manage all problems, hold contests.
- Markdown support.

## Requirements

Browser:
- IE 8+
- Chrome 30+
- Firefox 14+
